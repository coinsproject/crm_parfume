# Перезапуск после обновления

Если после обновления ничего не изменилось, выполните принудительную пересборку:

## Быстрый способ

```bash
cd /opt/crm_parfume
sudo docker-compose down
sudo docker-compose build --no-cache
sudo docker-compose up -d
sudo docker-compose exec crm alembic upgrade head
```

## Или используйте обновленный скрипт

```bash
cd /opt/crm_parfume
sudo git pull origin main
sudo ./update.sh
```

Обновленный скрипт теперь автоматически пересобирает образ при обновлении кода.

## Проверка версии

После перезапуска проверьте версию в интерфейсе:
- Откройте http://your-server:8000
- Версия должна быть **1.2.0** (внизу страницы или в настройках)

## Если версия не обновилась

```bash
# Проверьте логи
sudo docker-compose logs crm --tail=50

# Проверьте, что контейнер использует новый образ
sudo docker-compose ps
sudo docker inspect parfume-crm | grep Image

# Принудительно пересоберите и перезапустите
sudo docker-compose down
sudo docker-compose build --no-cache
sudo docker-compose up -d
```
